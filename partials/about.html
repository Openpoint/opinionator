The Opinionator {{slogan}} since 2014

The best unscientific research and a passionate belief in triviality has led to to 'The Opinionator' - an internet opinion hype machine.

It strives to answer two fundamental questions:

  1. What do you think about this
  2. What do others think about that

With these existential issues addressed, we will be ready for the return of our Llama overlords.

But seriously:

This is an Alpha, test release web-app built using Angular JS. I am building it in my spare time and will be releasing it under a GNU GPL license in due time.

The design intent is that it will be extremely easy to deploy on a standard Apache webserver so that everybody can start creating opinion streams.
